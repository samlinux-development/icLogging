info About IC Decentralised Audit

description Overview

IC Decentralised Audit is an application for logging and auditing data on the Internet Computer (IC) with a Motoko backend and Angular 21 frontend. It demonstrates how to build decentralized applications using immutable data structures.

Note: In a modified version, the add functionality can also be performed by other agents besides the web UI, such as Node.js or Go agent services running elsewhere. The backend API is accessible from any IC-compatible agent.

star Features
  • check_circle Immutable log entries stored on the Internet Computer
  • check_circle Material Design 21 UI components
  • check_circle Real-time log viewing with sorting and filtering
  • check_circle Add logs with different levels (INFO, WARN, ERROR)
  • check_circle Detailed audit entry view
  • check_circle Persistent storage using Motoko's immutable Map
